About Agile Poker

Agile Poker is a lightweight, real-time planning poker application built to help agile and scrum teams estimate story points collaboratively.

Purpose

This application is intended solely for professional software development teams to facilitate sprint planning and estimation sessions.

Security & Privacy

  • No user tracking
  • No ads
  • No analytics scripts
  • No cookies
  • No data is stored permanently

Technology

  • Angular frontend
  • Spring Boot backend
  • WebSocket (STOMP) for real-time updates

Use Case

Agile Poker is commonly used during sprint planning meetings to estimate user stories using Fibonacci-style point values.
